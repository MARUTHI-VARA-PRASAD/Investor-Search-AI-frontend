This project is a simple frontend application that allows users to search
for top investors based on sector and country using an AI API.

-- Features --
- Input sector and country
- AI-generated list of 3 investors
- One-line reasoning for each investor
- Clean and simple UI

-- Tech Stack --
- HTML
- CSS
- JavaScript
- OpenAI Chat Completion API

-- How to Run --
1. Clone the repository
2. Open `index.html` in a browser
3. Add your OpenAI API key in `script.js`

-- API Used --
OpenAI ChatGPT API (gpt-3.5-turbo)

-- API Key Setup --
Replace `This is the Api key's location which i could not paste due to security reasons  kindly understand` in `script.js` with your OpenAI API key.
